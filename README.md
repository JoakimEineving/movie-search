<h2>
:gear: How To Use
</h2>

```bash
# Clone this repository
$ git clone https://github.com/JoakimEineving/movie-search.git

# Navigate into the repository
$ cd movie-search

# Create a file in the root directory called '.env' and enter:
$ REACT_APP_API_KEY="YOUR_OMDB_API_KEY"

# Install dependencies
$ npm i

# Run the app and enjoy
$ npm start
```
